# reading-log

읽은 책의 정보, 메모, 인상 깊은 문장, 독후감을 정리하는 개인 독서 기록 저장소입니다.

---

## Overview

이 저장소에는 읽은 책을 한 권당 하나의 Markdown 파일로 기록합니다.

각 책 파일에는 보통 아래 내용을 포함합니다.

- 기본 정보
- 읽은 기간 및 완료일
- 한줄평
- 핵심 내용 정리
- 인상 깊은 문장
- 독후감
